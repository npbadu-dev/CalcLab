# CalcLab project - all mini project in one site

## What is CalcLab?

- CalcLab is a site, where you can find every project that i developed in the phase of learning javascript.

## Who is it for?
- It is for every fresher developer, who are new in this field and want to build a real life project, then can use it as an inspiration and start building by themselves.

## Why am I building it?
- I am building this because I want to list all my project that i develop while learning.
- Instead of creating many different repositories or folder for each mini project it became very messy, so I decide to create a website where I can showcase all my project which help me to manage and can also be helpful for other developer.

## What problem does it solve?
- This help me to manage all my practice project in one place, so it is very helpful for revising or reviewing my project.
- This solve, the messy file problem.

## What are the MVP features?
- This project is not a very functional fully-functional project, this is only my portfolio type project of my practice project.
- The MVP features are:
1. search to find project
2. dark and light toggle.

## What features can wait until version 2?
- At the beginning I will only make it showcasing, I will later add a functional activities, like searching.
